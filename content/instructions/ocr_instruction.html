Acceptance criteria:

Attention: Photos that do not meet any of the requirements will be rejected and not be paid!

1. Usually the number of a barcode is separated in groups, please divide each group with one and only one space symbol. 

Example №1:


For the barcode above the correct text is "1 234567 890128 >", because the number is divided into for groups: "1", "234567", "890128", and ">".

Example №2:


For the barcode above the correct text is "4820287750783", because the number is not separated into groups.

2. Use only digital symbols to describe digits. 

Example №1:


For the barcode above the correct text is "1 234567 890128 >", alternative ways to descibe digits are not acceptable.
For instance, such unacceptable way is using of numerals: one, tow, three, four, ...

3. Please, do not type anything except a barcode number.

Example №1:


For the barcode above the correct text is "4 891668 328881", the rest text of the photo is not acceptable.
